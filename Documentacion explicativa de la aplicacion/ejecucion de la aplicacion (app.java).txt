Detalle de lo que hace app.java:

Se importan las clases Estudiante, Hogwarts, ProcesadorArchivoCsv, ArrayList y Scanner.
Se declara el objeto miEscanner de la clase Scanner para recibir entrada del usuario.
Se muestra la interfaz de usuario para pedir el nombre del archivo CSV a procesar.
Se lee el nombre del archivo CSV del usuario utilizando miEscanner.nextLine().
Se crea un objeto ProcesadorArchivoCsv con el nombre de archivo ingresado.
Se llama al método procesarArchivoConValidacion() del objeto ProcesadorArchivoCsv para leer el archivo CSV y devolver una lista de objetos Estudiante validados.
Se muestra al usuario la cantidad de estudiantes que se leyeron correctamente del archivo CSV.
Se crea un objeto Hogwarts.
Se recorre la lista de estudiantes y se llama al método agregarEstudiante() del objeto Hogwarts para agregar cada estudiante a la lista de estudiantes.
Se muestra al usuario la cantidad de estudiantes en cada casa de Hogwarts llamando al método getCantidadEstudiantes() de cada casa del objeto Hogwarts.
En general, el archivo app.java se encarga de procesar la entrada del usuario, leer y validar los datos del archivo CSV, y mostrar la salida al usuario. También utiliza la clase Hogwarts para mantener una lista de estudiantes organizados por casa.
